# Vastpy Overview

## VMS REST API

The VMS REST API is a programmatic interface that allows you to manage your VAST cluster remotely. It provides access to a wide range of functionalities that are also available through the VAST Web UI (VMS GUI) and CLI. 

The VMS REST API can be accessed at `https://<VMS_VIP>/docs/index.html`.

## Vastpy

The vastpy package is a Python SDK to the VMS (VAST Management System) REST API.

This section will contain examples of vastpy.
